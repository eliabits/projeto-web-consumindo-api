# 🎣 Pesca Fácil

## 📖 Sobre o projeto

O **Pesca Fácil** é uma página web desenvolvida como atividade da disciplina de Desenvolvimento Web. O objetivo do projeto é consumir uma API pública utilizando a função **fetch()** do JavaScript, exibir os dados de forma dinâmica e apresentar uma interface responsiva utilizando **Tailwind CSS**.

A aplicação simula uma loja virtual de artigos de pesca, exibindo produtos em formato de cartões (cards) contendo imagem, nome, preço e uma breve descrição.

---

## 🚀 Tecnologias utilizadas

* HTML5
* css
* JavaScript (ES6)
* Fetch API
* Tailwind CSS (via CDN)

---

## 🌐 API utilizada

O projeto utiliza uma API pública para obter os dados dos produtos.

* Fake Store API

A API foi escolhida por ser gratuita, pública e não exigir autenticação (token), facilitando seu uso em projetos acadêmicos.

---

## ▶️ Como executar

1. Faça o download ou clone este repositório.
2. Abra o arquivo `index.html` em um navegador.
3. Clique no botão **Carregar Produtos**.
4. Os produtos serão carregados automaticamente pela API.

Não é necessário instalar dependências ou configurar servidor.

---

## 🛠️ Decisões técnicas

Durante o desenvolvimento foram adotadas algumas decisões para manter o projeto simples e compatível com os conteúdos estudados em sala de aula:

* Utilização apenas de HTML, JavaScript e Tailwind CSS.
* Consumo da API utilizando **fetch()**.
* Manipulação do DOM com JavaScript puro.
* Código dividido em arquivos separados (`index.html`, `script.js` e `style.css`) para facilitar a organização.
* Interface responsiva utilizando apenas classes do Tailwind CSS.
* Estrutura simples para facilitar a compreensão e manutenção.

---

## ⚠️ Limitações

Este projeto possui algumas limitações por ser uma aplicação demonstrativa:

* Os produtos são fornecidos por uma API pública e não representam uma loja real de pesca.
* O botão **Comprar** não possui funcionalidade.
* Não existe sistema de login.
* Não existe carrinho de compras.
* Não há sistema de busca ou filtros de produtos.
* Caso a API fique indisponível, os produtos não poderão ser carregados.

---

## 📚 Objetivo acadêmico

Este projeto foi desenvolvido com fins exclusivamente acadêmicos para demonstrar a utilização de:

* consumo de APIs públicas;
* uso da função **fetch()**;
* manipulação do DOM;
* criação de interfaces responsivas com Tailwind CSS;
* organização básica de um projeto web.
